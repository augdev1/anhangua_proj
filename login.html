# Anhangá — Plataforma de Monitoramento de Alertas Ambientais com IA Este projeto reúne backend (FastAPI) e frontend (HTML + Tailwind + Leaflet) para monitorar alertas ambientais na Amazônia, agora com **inteligência artificial para análise visual em tempo real**. --- ## 🌎 Objetivo do projeto O sistema foi construído para demonstrar uma solução integrada que: 1. consulta múltiplas fontes públicas de dados; 2. unifica e normaliza os alertas; 3. entrega visualização geográfica e dashboards; 4. aplica **IA para classificação visual do ambiente**; 5. registra evidências para auditoria. --- ## 🚀 Novidade: Monitoramento com IA O projeto agora inclui um módulo de visão computacional com: * Integração com **Teachable Machine** * Modelo exportado em **Keras (`keras_Model.h5`)** * Classificação em tempo real via webcam * Detecção automática de: * 🌳 Área preservada * 🔥 Área queimada --- ## 🧠 Arquitetura Atualizada ### Novos componentes #### 📷 `monitoramento.html` * Tela de análise em tempo real * Webcam ativa automaticamente * Classificação contínua com IA * Feedback visual: * status dinâmico * barra de confiança * histórico de eventos --- ## ⚙️ Dependências Atualize seu `requirements.txt`: ```txt requests python-dotenv scikit-learn numpy fastapi uvicorn tensorflow==2.21.0 opencv-python pillow ``` --- ## 📁 Estrutura adicional ```bash /model/ └── keras_Model.h5 # modelo exportado do Teachable Machine ``` --- ## ▶️ Execução ### 1. Instalar dependências ```bash pip install -r requirements.txt ``` --- ### 2. Rodar sistema completo ```bash python app.py ``` O sistema irá: * ✔ iniciar backend * ✔ iniciar frontend * ✔ verificar IA (TensorFlow + modelo) --- ## 📷 Monitoramento em tempo real Acesse: ``` http://127.0.0.1:5500/monitoramento.html ``` Funcionalidades: * câmera ativada automaticamente * detecção contínua * status em tempo real: * 📡 Conectando câmera... * ❌ Câmera desconectada * 🔥 Área queimada * 🌱 Área preservada --- ## 🔗 Navegação do sistema * `mapa.html` → visualização geográfica * `insights.html` → indicadores * `monitoramento.html` → IA em tempo real 🎥 --- ## 🧪 Fluxo atualizado 1. usuário acessa o sistema 2. navega entre mapa e insights 3. acessa monitoramento ao vivo 4. câmera captura imagem 5. modelo IA classifica o terreno 6. resultado é exibido instantaneamente --- ## ⚠️ Pontos de atenção * O modelo `.h5` deve estar em `/model/` * Navegador precisa permitir acesso à câmera * TensorFlow pode ser pesado em máquinas mais fracas * Classificação depende da qualidade do treinamento no Teachable Machine --- ## 📈 Próximos passos (evolução) * integração da IA com backend (API) * salvar histórico de detecções no banco * alertas automáticos (push/email) * deploy em nuvem * uso com câmeras reais (IoT) --- ## ✅ Conclusão O projeto evoluiu de um sistema de monitoramento passivo para uma plataforma **ativa com inteligência artificial**, capaz de analisar o ambiente em tempo real. 👉 Agora você tem: * dados + visualização + IA + automação ---
